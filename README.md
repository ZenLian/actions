# 🤖 Actions

Take use of github actions to *薅羊毛*.

## 🚀 Setup secrets

- `${{ secrets.EVER_PHOTO_DATA }}`: EverPhoto account info, in format `mobile1,passwd1;mobile2,passwd2`
- `${{ secrets.WX_CORP_ID }}`
- `${{ secrets.WX_APP_ID }}`
- `${{ secrets.WX_APP_SECRET }}`
